# AFFAN Checkpoint Teleporter

Auto teleport ke setiap checkpoint sampai summit untuk Roblox obby/tower games.

## Features

- 🎯 Auto-scan checkpoints di workspace
- 🚀 Sequential teleport CP 1 → summit
- 🔄 Loop mode (auto restart setelah summit)
- ⏸️ Pause/Resume control
- ⚙️ Configurable delay (0.1-3.0s)
- 📊 Real-time progress tracking
- 🎨 WindUI interface

## Installation

Copy dan paste ke executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/MiksuTrg/affan-checkpoint-teleporter/main/affan.lua"))()
```

## Usage

1. Execute script
2. Klik **SCAN CHECKPOINTS**
3. Klik **START TELEPORT**
4. Toggle **Loop Mode** untuk repeat

## Settings

- **Loop Mode**: Auto restart setelah summit
- **Delay Between TP**: 0.1-3.0 detik antar teleport
